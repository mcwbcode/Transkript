Fonctionnalités

Tout pour dicter avec efficacité

Transkript ne se contente pas de transcrire — il rend chaque dictée naturelle, rapide et intelligente.

⌨️
Raccourci universel
Appuyez sur Ctrl + → depuis n'importe quelle application Windows pour lancer la dictée instantanément.
⚡
Traitement ultra-rapide
Whisper tourne entièrement sur votre machine — transcription immédiate, sans latence réseau ni dépendance cloud.
🌍
100+ langues supportées
Dictez en français, anglais, espagnol, allemand, arabe, japonais et plus de 97 autres langues nativement.
🔒
Traitement 100 % local
Vos paroles ne quittent jamais votre PC. Le modèle IA Whisper tourne intégralement en local après le téléchargement initial.
📋
Collage automatique
Le texte transcrit est directement collé dans l'application active — sans copier-coller, sans manipulation supplémentaire.
📶
Fonctionne hors ligne
Après le téléchargement unique du modèle (~488 Mo), Transkript fonctionne sans aucune connexion internet.
🎙️
Dictée naturelle
Parlez normalement — Transkript gère les pauses, les reprises et produit un texte propre et immédiatement utilisable.
🛡️
Aucune donnée collectée
Vos dictées ne sont jamais envoyées à nos serveurs ni utilisées pour entraîner un quelconque modèle IA.
🚀
Zéro configuration
Installez, connectez-vous, dictez. Aucun paramétrage complexe — Transkript est opérationnel en quelques secondes.

Prêt en 4 étapes

👤

⬇️

Confidentialité

Conçu pour la vie privée

Vos données vous appartiennent. Transkript ne les collecte pas, ne les transmet pas, ne les exploite pas.

🛡️
Zéro rétention cloud
Aucune de vos dictées n'est envoyée sur nos serveurs ou ceux d'OpenAI. Tout reste sur votre machine, toujours.
🤖
Non entraîné sur vos données
Nous n'utilisons jamais vos dictées pour entraîner ou améliorer des modèles IA. Votre voix reste strictement privée.
💾
Historique stocké localement
Votre historique de transcriptions est stocké exclusivement sur votre PC. Vous gardez un contrôle total sur vos données.
