Comment dicter

Transkript fonctionne dans n'importe quelle application, sur Windows et macOS.

⌨️
Raccourcis clavier

Placez votre curseur là où vous voulez écrire, puis :
Windows : Ctrl + → pour commencer à dicter.
macOS : ⌥ Space pour commencer à dicter.

🔒
100 % privé

Vos dictées sont traitées localement sur votre machine par Whisper d'OpenAI. Aucune donnée n'est envoyée vers nos serveurs ou le cloud.
